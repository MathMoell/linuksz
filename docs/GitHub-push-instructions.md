# Kui soovid teha GitHubi repo (täpsem)
1. Loo uus repo GitHubis (nimi `linux-log-lab`).
2. Laadi siinsed failid üles.
3. Lisa README-sse lühike selgitus mida tegid ja lisainfo: kuupäev, kasutatud VM- ja Ubuntu versioon.
4. Soovituslik commit-sõnum: `feat: lisa labori juhend, konfiguratsioonid ja skriptid` ja PR kui vajab hindamist.
